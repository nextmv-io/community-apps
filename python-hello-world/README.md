# Nextmv Python Hello World

Simple example for running a Python application on the Nextmv Platform.

1. Install packages.

    ```bash
    pip3 install -r requirements.txt
    ```

1. Run the app.

    ```bash
    cat input.json | python3 main.py
    ```
