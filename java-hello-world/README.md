# Nextmv Java Hello World

Simple example for running a Java `Maven` application on the Nextmv Platform.

1. Generate a `main.jar`.

    ```bash
    mvn package
    ```

2. Run the app.

    ```bash
    cat input.json | java -jar main.jar
    ```
