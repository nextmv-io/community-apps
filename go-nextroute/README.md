# Nextmv Go Nextroute

Example for running a Go application on the Nextmv Platform using the
`Nextroute` vehicle routing problem solver.

1. Run the app.

    ```bash
    go run . -runner.input.path input.json \
      -runner.output.path output.json -solve.duration 10s
    ```
